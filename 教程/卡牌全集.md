# 卡牌全集

## 卡牌系统说明

### 基础属性
每张卡牌拥有以下四项核心属性：
| 属性 | 说明 |
|------|------|
| **生命值 (Health)** | 卡牌的耐久度，归零后卡牌死亡 |
| **攻击力 (Power)** | 攻击时造成的伤害值，或特殊效果的强度 |
| **速度 (Speed)** | 决定卡牌行动顺序，速度越高越先行动 |
| **信仰消耗 (Faith Cost)** | 打出该卡所需的信仰值资源 |

### 特殊效果一览
| 效果名称 | 效果说明 |
|----------|----------|
| **近战攻击 (Melee Attack)** | 需要部署在**前排**才能发动，按位置优先级攻击对方单位 |
| **远程攻击 (Ranged Attack)** | 需要部署在**后排**才能发动，优先攻击对方后排敌人 |
| **减伤 (Tough)** | 受到伤害时，伤害值减少等同于自身攻击力的数值 |
| **治疗 (Healer)** | 需要部署在**后排**才能发动，回复前排友方单位的生命值（回复量=自身攻击力） |
| **守护 (Guardian)** | 需要部署在**前排**才能发动，保护相邻的前排友方单位，替其承受部分伤害 |
| **近战范围攻击 (Melee Area Attack)** | 对敌方所有**前排**单位造成伤害 |
| **远程范围攻击 (Ranged Area Attack)** | 对敌方所有**后排**单位造成伤害 |
| **全体攻击 (All Area Attack)** | 对所有敌方单位造成伤害 |
| **星 (Star)** | 特殊效果（待补充） |
| **月 (Moon)** | 特殊效果（待补充） |

### 战斗流程
1. **玩家行动阶段**：玩家消耗信仰值部署卡牌到战场
2. **敌人行动阶段**：AI 控制敌人部署卡牌
3. **卡牌行动阶段**：所有场上卡牌按 **速度从高到低** 依次执行攻击或技能
4. **回合结束**：结算持续效果，进入下一回合

## 战术建议

1. **前排布局**：优先部署拥有 **近战攻击** 或 **减伤** 的单位（如金刚力士、文殊菩萨剑、佛首），它们负责抵挡敌人攻击。
2. **后排输出**：在后排放置 **远程攻击** 单位（如飞天、文殊菩萨经书、莲花化身），安全地造成伤害。
3. **速度为王**：高速单位（如飞天 Speed=6）可以抢先击杀敌方关键单位。
4. **信仰管理**：合理分配信仰值，不要一次性用光，留有余力应对后续回合。
